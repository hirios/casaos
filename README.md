### Como usar:


Dê permissões: `chmod +x hiriosDDclone.sh`

Execute como root: `./hiriosDDclone.sh <dispositivo> <caminho_saida>`
```
./hiriosDDclone.sh /dev/mmcblk0 /mnt/hdd/pasta/filename.img
```
